What is Arctext?

Arctext.js is a jQuery plugin that is based off of Lettering.js, which is also a jQuery plugin. While the use of CSS3 lets us arrange and rotate letters on a curved path going up or down, it is quite complicated to do so with CSS3 alone. With Arctext, designers/developers can use it to place letters on a arc type of path with a equal direction, rotation and radius of the path. It sounds complicated but it is quite simple using the Arctext plugin to place each letter on the arc type of path with the letters shown with an equal amount of space.

How is Arctext used?

Arctext is used by using width, rotation, and radius of text. Direction is also used in whether or not the text should be displayed going up or down in a arc type manner. The path of the text is based on a imaginary circle in which the path follows. In order for the letters to be displayed, the rotation and space of the letters are based on the inputs of direction, width, rotation and radius of which the CSS3 coding and jQuery coding defines. Arctext will calculate the code, and then place the letters equally along the path.

Why I choose Arctext

I chose Arctext because I think it looks more professional in webpages and give the webpages a nice clean look. Also, it gives a sense of life to the layout of the page and is more stylish from the norm. By norm, I mean the simple left, right or center alignment of the text displayed on the web. Using Arctext is useful by giving words, logos, or phrases characteristics and uniqueness that would appeal more to the viewer, instead of the same alignments of text that we see throughout a lot of webpages.

return false;

return false;
